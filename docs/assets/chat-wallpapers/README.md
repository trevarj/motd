# IRC wallpaper system

The canonical wallpaper masks live in `app/src/main/assets/chat-wallpapers/` and are generated by
`tools/wallpapers/generate-wallpapers.mjs`. They are transparent, monochrome, seamless 512×512 SVG
tiles. The app combines each mask with a theme-derived gradient and user-controlled intensity.

Regenerate and verify from the pinned Nix development shell:

```sh
nix develop -c node tools/wallpapers/generate-wallpapers.mjs
nix develop -c node tools/wallpapers/generate-wallpapers.mjs --check
```

The generator intentionally uses curated placements rather than random scatter. Motifs crossing a
tile edge are paired exactly 512 units apart, and runtime loading accepts only the constrained SVG
subset covered by `WallpaperAssetTest`.
