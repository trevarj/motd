# Chat sound assets

MOTD's send and receive cues are original synthesized PCM samples generated by
`tools/chat-sounds/generate-chat-sounds.mjs`. Both use the same short pop-click timbre and envelope.
Send is voiced at C#6 and receive at F#6, an ascending equal-tempered perfect fourth that makes the
two actions form a subtle cadence. Receive is slightly louder for clarity. They follow familiar
messaging-app acoustic roles without copying audio from Telegram or another application.

Regenerate and verify them from the pinned development shell:

```sh
nix develop -c node tools/chat-sounds/generate-chat-sounds.mjs
nix develop -c node tools/chat-sounds/generate-chat-sounds.mjs --check
```

Both resources are mono, 48 kHz, 16-bit WAV files with click-free fades. Receive peaks at
−6 dBFS; send peaks at −9 dBFS so rapid messages remain unobtrusive. They are part of MOTD and
distributed under the repository's GPL license.
